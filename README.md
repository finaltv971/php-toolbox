# Boîte à outils PHP

Scripts PHP et Laravel, helpers, extraiteurs et utilitaires réutilisables en tout genre pour le backend au quotidien.

## Objectif

Ce référentiel est une boîte à outils publique pour les développeurs qui veulent des exemples backend prêts à l'emploi ou faciles à adapter en PHP et Laravel.

Il contient :

- Scripts PHP autonomes

- Extraits et utilitaires Laravel

- Fonctions d'aide

- Exemples de service

- Commandes de console

- Emplois

- Exemples d'intégration

- Modèles réutilisables pour les tâches backend communes

## Philosophie du référentiel

Ce projet est destiné à rester :

- Simple

- Réutilisable

- Conscient du cadre

- Public-sûr

- Facile à parcourir

## Règles

- Pas de code commercial spécifique au client

- Pas de secrets, de clés API, de jetons ou d'informations d'identification

- Pas de vidages de production ou de journaux sensibles

- Pas de points de terminaison privés codés en dur

- Les exemples doivent utiliser uniquement de fausses valeurs

- Le code doit être compréhensible et adaptable
