# Boîte à outils 

Scripts, code, helpers, extraiteurs et utilitaires réutilisables en tout genre pour le quotidien.

## Objectif

Ce référentiel est une boîte à outils publique pour les développeurs qui veulent des exemples prêts à l'emploi ou faciles à adapter a vos besoins.

Il contient :

- Scripts et exemple de code

- Extraits et utilitaires en php/js/swift/python

- Fonctions d'aide

- Exemples de service

- Commandes de console

- Emplois

- Exemples d'intégration

- Modèles réutilisables pour les tâches backend/frontend/mobile/ai communes

## Philosophie du référentiel

Ce projet est destiné à rester :

- Simple

- Réutilisable

- Conscient du cadre

- Public-sûr

- Facile à parcourir

## Règles

- Pas de code commercial spécifique au client

- Pas de secrets, de clés API, de jetons ou d'informations d'identification

- Pas de vidages de production ou de journaux sensibles

- Pas de points de terminaison privés codés en dur

- Les exemples doivent utiliser uniquement de fausses valeurs

- Le code doit être compréhensible et adaptable
